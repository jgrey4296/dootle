Introduction
------------

Dootle is an extension library for :ref:`doot`. See there for general usage guidance.
It adds specific tasks and actions that don't need to be part of the main library.


Repo and Issues
---------------

The repo for doot can be found `here <https://github.com/jgrey4296/dootle>`_.

If you find a bug, bug me, unsurprisingly, on the `issue tracker <https://github.com/jgrey4296/dootle/issues>`_.
